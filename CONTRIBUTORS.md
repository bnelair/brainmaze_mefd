## Contributing

We welcome contributions from the community! If you contribute to brainmaze_mefd, your name will be listed here. Please see [CONTRIBUTING.md](CONTRIBUTING.md) for contribution guidelines.

### How to Get Listed as a Contributor

1. Fork the repository
2. Make your changes (code, docs, examples, etc.)
3. Submit a pull request with a description of your contribution
4. Once accepted, you'll be added to this list

### Recognition Levels

- **Code Contributors**: Contributed code, bug fixes, or features
- **Documentation Contributors**: Improved documentation or examples
- **Community Contributors**: Provided feedback, reported bugs, or helped other users
- **Sponsor/Supporter**: Provided financial or infrastructure support
